# Third-party inputs

The `data/frozen_third_party_inputs/` directory contains exact frozen public catalogue files used to establish a reproducible analysis state. These files are included for verification and do not supersede their source archives.

Required citations and official locations are listed in `provenance/frozen_input_checksums.csv` and `provenance/catalogue_provenance.csv`. Users must comply with the original archive terms when redistributing or reusing these files.
