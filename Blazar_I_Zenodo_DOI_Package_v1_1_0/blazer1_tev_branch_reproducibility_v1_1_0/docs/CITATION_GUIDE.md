# Citation guide

1. Exact reproducibility uchun **version-specific Zenodo DOI** ni cite qiling.
2. Associated manuscript mavjud bo‘lgach, uni ham cite qiling.
3. Original BlazEr1, eROSITA, Fermi-LAT/4LAC va TeVCat resurslarini `provenance/catalogue_provenance.csv` bo‘yicha cite qiling.
4. Faqat Zenodo concept DOI bilan cheklanmaslik tavsiya etiladi; exact release uchun version DOI kerak.
