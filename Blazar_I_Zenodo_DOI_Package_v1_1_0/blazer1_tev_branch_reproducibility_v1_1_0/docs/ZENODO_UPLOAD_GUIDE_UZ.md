# Zenodo'ga yuklash bo‘yicha ko‘rsatma

1. Zenodo’da **New upload** oching.
2. Resource type sifatida **Software** ni tanlang.
3. `Blazar_I_Zenodo_DOI_Package_v1_1_0.zip` faylini yuklang.
4. `metadata/zenodo_deposit_metadata.json` ichidagi title, description, keywords va version qiymatlarini formaga ko‘chiring.
5. Creatorlar, affiliation va ORCIDlarni yakuniy manuscript bilan solishtirib tekshiring.
6. **Get a DOI now** orqali version DOI’ni oldindan rezerv qiling.
7. Rezerv qilingan DOI’ni manuscript Data Availability bo‘limiga kiriting.
8. License sifatida **CC BY 4.0** ni tanlang. Kodning MIT litsenziyasi va uchinchi tomon fayllari haqidagi istisnolar `LICENSES.md` da berilgan.
9. Preview’da ZIP nomi, version, creator order, description va file count’ni tekshiring.
10. Journal DOI chiqqach, related identifier sifatida `isSupplementTo` munosabatini qo‘shing.
