# Data dictionary

The release contains two machine-readable result families.

## Core BlazEr1 catalogues

- `full_catalogue_column_dictionary.csv`: all columns in the full active catalogue.
- `core_catalogue_column_dictionary.csv`: compact active-catalogue fields recommended for routine use.

## 4LAC external-transfer catalogues

- `4lac_transfer_score_catalogue_dictionary.csv`: compact 25-column frozen transfer-score catalogue.
- `4lac_transfer_cohort_dictionary.csv`: complete 4LAC-exclusive cohort, including overlap, label, and evaluability audit fields.

The compact transfer-score catalogue is recommended for scientific reuse. The full cohort file preserves match geometry and audit lineage.
